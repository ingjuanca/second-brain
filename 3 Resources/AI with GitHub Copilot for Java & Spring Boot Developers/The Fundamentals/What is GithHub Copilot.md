**¿Qué es GitHub Copilot?**

En esta lección aprenderás qué es GitHub Copilot.

GitHub Copilot es un programador asistente basado en inteligencia artificial que puede trabajar con cualquier lenguaje de programación, scripts de DevOps o incluso ayudarte con comandos desde la línea de comandos.

Fue desarrollado en colaboración con OpenAI y está impulsado por OpenAI Codex.

Codex es un modelo de IA generativa entrenado con un enorme conjunto de datos que incluye código disponible en Internet y también texto en lenguaje natural.

Se integra en la mayoría de los IDEs populares actuales.

Copilot puede analizar el contexto del código observando clases, funciones y variables, y proporcionar autocompletado y sugerencias de código.

Puede generar fragmentos de código o funciones completas basándose en comentarios o simplemente a partir del nombre de una función.

También puede documentar el código y explicar código complejo, lo cual es muy útil cuando se trabaja en un proyecto existente o cuando se está aprendiendo una nueva librería.

Copilot Chat funciona de manera similar a ChatGPT, pero está especializado en responder problemas relacionados con programación.

Puede ayudarte a depurar errores, darte pasos para resolver un problema específico o incluso generar un programa completo a partir de un prompt.

# ✅ **Resumen muy completo del documento**

El documento presenta a **GitHub Copilot** como una herramienta de inteligencia artificial diseñada para actuar como un _programador asistente_ dentro de los entornos de desarrollo.

Copilot fue desarrollado por GitHub en colaboración con OpenAI y se basa en **OpenAI Codex**, un modelo de IA entrenado con grandes volúmenes de código y texto en lenguaje natural. Gracias a este entrenamiento, es capaz de comprender tanto instrucciones escritas en lenguaje humano como el contexto del código fuente.

Una de sus principales características es su **integración directa con los IDEs más populares**, lo que le permite analizar en tiempo real clases, funciones y variables para ofrecer **sugerencias automáticas, autocompletado inteligente y generación de código**.

GitHub Copilot puede:

- Generar fragmentos de código o funciones completas a partir de:
    
    - Comentarios en lenguaje natural.
        
    - El nombre de una función.
        
    - El contexto del archivo.
        
- Ayudar a documentar código automáticamente.
    
- Explicar código complejo, facilitando el entendimiento de proyectos existentes o nuevas librerías.
    
- Apoyar tareas de depuración y resolución de problemas.
    

Además, el documento introduce **Copilot Chat**, una funcionalidad similar a ChatGPT, pero enfocada específicamente en desarrollo de software. Esta permite:

- Hacer preguntas técnicas.
    
- Pedir ayuda para encontrar errores.
    
- Solicitar pasos para resolver problemas.
    
- Generar programas completos basados en descripciones.
    

En conjunto, GitHub Copilot se presenta como una herramienta que **aumenta la productividad del desarrollador**, reduce el tiempo de escritura de código, facilita el aprendizaje y mejora la comprensión de sistemas existentes.
