En esta lección te mostraré cómo configurar GitHub Copilot en Visual Studio Code.

Para hacerlo, necesitas tener una suscripción a GitHub Copilot.

Inicia sesión en github.com, ve a tu perfil, haz clic en Copilot y puedes obtener una suscripción individual.

Si no tienes una suscripción Enterprise proporcionada por tu empresa, el primer mes de la suscripción individual es gratuito.

Tendrás acceso gratuito, lo cual es suficiente para que puedas dominar GitHub Copilot.

Para GitHub Copilot y GitHub Copilot Chat existen **dos extensiones separadas**.  
Con suerte, en el futuro ambas estarán unificadas en una sola extensión.

Selecciona GitHub Copilot Chat e instala esa extensión también.

Una vez instaladas, si es necesario, reinicia el IDE.

El sistema te pedirá que inicies sesión con tu cuenta de GitHub.

Ingresa tus credenciales de GitHub y estarás listo para comenzar.

# ✅ **Resumen muy completo del documento**

El documento explica paso a paso cómo **configurar GitHub Copilot en Visual Studio Code**.

Primero, se aclara que es necesario contar con una **suscripción a GitHub Copilot**. Esta puede obtenerse desde github.com, accediendo al perfil del usuario y seleccionando la opción Copilot. Si el usuario no cuenta con una suscripción Enterprise, GitHub ofrece un **período de prueba gratuito de un mes** para la suscripción individual, suficiente para aprender y dominar la herramienta.

Luego, el documento destaca que actualmente **GitHub Copilot y GitHub Copilot Chat son extensiones independientes**, por lo que es necesario instalar ambas desde el marketplace de Visual Studio Code. Se menciona que en el futuro podrían unificarse en una sola extensión.

Después de instalar las extensiones, puede ser necesario **reiniciar el IDE**. Al abrirlo nuevamente, Visual Studio Code solicitará iniciar sesión con una cuenta de GitHub.

Finalmente, tras completar el inicio de sesión, el entorno quedará listo para usar GitHub Copilot y Copilot Chat.
