A lo largo del curso estaré usando Visual Studio Code.

Esto se debe a que en Visual Studio Code tendremos acceso tanto a GitHub Copilot como a Copilot Chat, incluso con una suscripción individual de GitHub Copilot. Con la suscripción individual se tiene acceso tanto a Copilot como a Copilot Chat, lo cual es muy útil.

Esto no ocurre con IntelliJ IDEA, al menos hasta febrero de 2024.

Puedes instalar fácilmente GitHub Copilot yendo al marketplace de IntelliJ IDEA.

Después de instalarlo, reinicia el IDE.

Una vez lo reinicies, verás Copilot y Copilot Chat en el lado derecho.

Copilot estará accesible, pero Copilot Chat mostrará un mensaje indicando que necesitas una suscripción Enterprise de Copilot. Con una suscripción individual no podrás acceder a Copilot Chat en IntelliJ IDEA (al menos hasta febrero de 2024). Sin embargo, esto no sucede en Visual Studio Code.

Esa es la razón por la cual se usa Visual Studio Code a lo largo del curso.

Si tienes acceso a Copilot Chat (es decir, si cuentas con una suscripción Enterprise), eres bienvenido a usar IntelliJ u otro IDE donde tengas acceso tanto a Copilot como a Copilot Chat.

# ✅ **Resumen muy completo del documento**

El documento explica la **elección del entorno de desarrollo (IDE)** para trabajar con GitHub Copilot durante el curso, destacando las diferencias prácticas entre Visual Studio Code e IntelliJ IDEA.

Se aclara que el curso utiliza **Visual Studio Code** porque, con una **suscripción individual de GitHub Copilot**, es posible acceder tanto a **Copilot** como a **Copilot Chat**, lo cual amplía significativamente las capacidades de asistencia (autocompletado, generación de código y chat especializado para programación).

En contraste, en **IntelliJ IDEA**, aunque GitHub Copilot se puede instalar fácilmente desde el marketplace y funciona para sugerencias de código, **Copilot Chat no está disponible para usuarios con suscripción individual** (al menos hasta febrero de 2024). Para usar Copilot Chat en IntelliJ, se requiere una **suscripción Enterprise**.

Por esta razón, el autor decide trabajar con Visual Studio Code durante el curso, garantizando que todos los participantes tengan acceso completo a las funcionalidades de Copilot.

Finalmente, se aclara que quienes cuenten con una suscripción Enterprise pueden usar IntelliJ u otros IDEs, siempre y cuando tengan acceso tanto a Copilot como a Copilot Chat.
