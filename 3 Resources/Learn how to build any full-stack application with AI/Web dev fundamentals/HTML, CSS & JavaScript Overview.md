> Así que comencemos ahora con los conceptos básicos de HTML, CSS y JavaScript.
> 
> Cada página web está construida con HTML, CSS o JavaScript, y puedes pensar en HTML como la estructura del sitio web o de la aplicación web. Y CSS es el estilo y diseño que se construye sobre esa estructura. Y luego, finalmente, JavaScript ayuda a añadir características y funcionalidades al sitio web o página web.
> 
> Si quieres pensar en esto en términos más simplificados, HTML es simplemente el esquema de un sitio web. Por ejemplo, si tenemos Facebook, tenemos un encabezado, tenemos el título del sitio web; todo eso es HTML. Y luego CSS va un paso más allá y le da estilo a todas esas estructuras. Para eso tenemos tamaños de fuente, colores de fondo. Y finalmente tenemos JavaScript, que añade animaciones al sitio, así como funcionalidad. De este modo, no es solo un sitio web estático con palabras en una página; serás capaz de interactuar con la interfaz de usuario (_UI_).
> 
> Estos tres lenguajes son los que todos los desarrolladores web deben aprender. Así que en esta sección repasaré los conceptos básicos y fundamentales de HTML, CSS y JavaScript. Pero más adelante, cuando trabajemos con React y Next.js, especialmente con la IA, estos combinarán los tres lenguajes independientes en una estructura más eficiente. Sin embargo, por ahora debes conocer los conceptos básicos para entender realmente lo que está sucediendo en el código.

## 💡 Ideas más relevantes

- **Los tres pilares del desarrollo web:** Cualquier página en internet se fundamenta en tres tecnologías esenciales: **HTML** (define el esquema y la estructura), **CSS** (aplica el diseño visual, fuentes y colores) y **JavaScript** (aporta dinamismo, animaciones y funciones interactivas).
    
- **Fin de la web estática:** La incorporación de JavaScript es el elemento clave que transforma un sitio plano de lectura en una interfaz interactiva con la que el usuario puede operar de forma directa.
    
- **Evolución hacia la unificación moderna:** Aunque tradicionalmente se estudian por separado como lenguajes obligatorios, herramientas avanzadas como **React** y **Next.js** permiten unificar estas tres capas en una estructura única y mucho más eficiente al trabajar en conjunto con la IA.
    
- **Necesidad de la base teórica:** A pesar de la simplificación que ofrecen los marcos de trabajo modernos y la automatización de la IA, poseer un conocimiento básico de estos pilares es indispensable para comprender la lógica detrás del código generado.
