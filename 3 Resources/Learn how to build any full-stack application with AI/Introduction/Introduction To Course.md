## 💡 Ideas más relevantes

- **Desarrollo guiado por IA:** El objetivo central del curso es enseñar a construir aplicaciones web completas (_full stack_) utilizando exclusivamente herramientas de inteligencia artificial.
    
- **Fundamentos técnicos necesarios:** Aunque se use IA, la primera parte del curso se enfoca en las bases de la programación para que el estudiante sea capaz de supervisar la generación de código y solucionar errores de manera eficiente.
    
- **Proyecto práctico como base:** El aprendizaje culminará con la creación de una aplicación CRUD (Crear, Leer, Actualizar y Borrar, por sus siglas en inglés), la cual funcionará como una plantilla o estructura base para futuros proyectos web.
